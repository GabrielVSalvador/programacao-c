# 🖥️ Projetos em Linguagem C

Este repositório contém meus estudos, algoritmos e estruturas de dados desenvolvidos em **Linguagem C**. Aqui documento meu aprendizado sobre manipulação de memória, ponteiros e lógica de baixo nível.

## 🚀 Tecnologias e Ferramentas
- **Linguagem:** C (Padrão C11/C17)
- **Compilador:** GCC (GNU Compiler Collection)
- **Ambiente:** Linux (Lubuntu)
- **Editor:** codeblocks

## 📂 Estrutura do Repositório
- `/exercicios`: Problemas de lógica e fixação.
- `/estruturas-dados`: Implementações de Filas, Pilhas e Listas Encadeadas.
- `/projetos`: Pequenos sistemas (ex: Cadastro de Clientes, Calculadoras).
